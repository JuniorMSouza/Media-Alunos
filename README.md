# 📚 Sistema de Cálculo de Média de Alunos

Projeto desenvolvido em **Python** com o objetivo de praticar **fundamentos de programação e lógica**, incluindo validação de dados, estruturas condicionais e funções.

---

## 📖 Sobre o Projeto

Este programa solicita o nome de um aluno e quatro notas, valida as entradas do usuário e calcula a média final do estudante.

Com base na média, o sistema informa se o aluno está:

- ✅ Aprovado
- ⚠️ Recuperação
- ❌ Reprovado

O sistema também impede erros comuns, como:

- Digitar letras no lugar de números
- Digitar notas menores que **0**
- Digitar notas maiores que **10**

---

## 🚀 Funcionalidades

✔ Solicitar nome do aluno  
✔ Solicitar quatro notas  
✔ Validação de entrada de dados  
✔ Tratamento de erro (`try / except`)  
✔ Cálculo de média automática  
✔ Classificação do aluno
